# Pneumonia Detection using CNN

## Project Description
This project aims to build a machine learning model that can detect pneumonia from chest X-ray images. Pneumonia is a serious lung disease, and early detection can help in faster treatment. We use a Convolutional Neural Network (CNN) to classify X-ray images as either **Normal** or **Pneumonia**.

---

## Objectives
- Build a deep learning model for medical image classification  
- Train the model using chest X-ray images  
- Evaluate model performance using accuracy and confusion matrix  
- Visualize results using graphs and plots  

---

## Dataset
We use a chest X-ray image dataset containing two classes:
- Normal  
- Pneumonia  

The dataset is divided into:
- Training set  
- Validation set  
- Test set  

---

## Methods
- Image preprocessing (resizing, normalization)  
- Convolutional Neural Network (CNN)  
- Model training and validation  
- Performance evaluation
